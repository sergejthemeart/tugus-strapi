# `@repo/design-system`

Shared Tailwind v4 theme package. It contains the source CSS variables and builds the CSS and editor configuration files used by the UI and Strapi admin.

The package lives in:

```text
packages/design-system
```

Use the [**Design System docs**](/docs/design-system) as the primary source for token, style, typography, and editor guidance.

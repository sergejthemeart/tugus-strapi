# find-skills

Find installable agent skills when the current skill set does not cover a task. It helps search the open skills ecosystem, verify candidate quality, and install a skill when the user wants it.

## Use it when

- You ask whether a skill exists for a task.
- You want to extend the agent with help for a repeated specialized task.
- A repeated task looks like it may already have an installable skill.

## What it helps solve

- Finds an existing skill for a repeated or specialized task.
- Helps decide whether to install a skill or handle the task directly.
- Gives users a short list of relevant options instead of a broad search dump.

Use it when you want to expand what the agent can do for future tasks.

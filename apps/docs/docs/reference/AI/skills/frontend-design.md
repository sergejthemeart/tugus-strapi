# frontend-design

Guide visual direction for new UI or substantial UI reshaping. It pushes the agent to make design choices that fit the subject instead of producing a generic layout.

## Use it when

- You are building a new page, app surface, or visual component.
- A UI needs stronger art direction, typography, layout, or copy.
- A previous design feels templated or visually flat.

## What it helps solve

- Gives a new interface a clearer visual identity.
- Improves layout, typography, color, motion, and copy choices.
- Helps move a page away from generic templates toward something suited to its audience.

Use it when the look and feel matter, not just the mechanics.
